# ソースコード構成定義およびモジュール仕様書

本書は、自動車保険見積システムのソースコード（バックエンド：Java/Spring Boot、フロントエンド：React/TypeScript）のディレクトリ構成、各モジュール（クラス・コンポーネント）の定義、およびアーキテクチャ設計方針についてまとめた交付文書（提出物）です。

---

## 1. アーキテクチャ設計基本方針

本システムは、保守性、拡張性、およびテストの容易性を最大化するため、関心の分離（Separation of Concerns）に基づき、明確なレイヤードアーキテクチャ（Layered Architecture）を採用しています。

```
[フロントエンド: SPA (React)]
       │ (HTTP REST API / JSON)
       ▼
[コントローラ層: Controller] ── 入力データの型検証・認証認可ハンドリング
       │
       ▼
[ビジネスロジック層: Service] ── 算定ロジックの適用、マスタデータ連携、キャッシュ制御
       │
       ▼
[データアクセス層: Repository / Entity / DTO] ── DB永続化・クエリ実行
```

## 5. 見積計算処理フロー (Flow Analysis)

ユーザーが見積作成ボタンを押下した際、内部では以下のシーケンスでビジネスロジックが実行されます。

```text
[Browser]             [QuoteController]             [QuoteService]            [Database / Cache]
    │                        │                            │                           │
    ├─ POST /api/quotes ────>│                            │                           │
    │  (QuoteRequest JSON)   │                            │                           │
    │                        ├─ @Valid DTO検証 ──────────>│                           │
    │                        │  (入力値のバリデーション)   │                           │
    │                        │                            ├─ applyMultipliers ───────>│
    │                        │                            │  (年齢/免許等のマスタ取得)  │ (インメモリキャッシュ)
    │                        │                            ├─ sumAdditions ──────────>│
    │                        │                            │  (特約等のマスタ取得)     │
    │                        │                            ├─ roundPremium             │
    │                        │                            │  (年額/月額の四捨五入計算)  │
    │                        │                            ├─ generateQuoteNo          │
    │                        │                            │  (見積番号の採番)         │
    │                        │                            ├─ buildQuoteEntity ────────┼── DB Insert
    │                        │                            │  (親子リレーション保存)    │
    │                        │<─ 201 Created 返却 ────────┤                           │
    │<─ 見積結果JSON ────────┤
```
