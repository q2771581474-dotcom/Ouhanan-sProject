# テスト仕様書兼テスト結果報告書

本書は、自動車保険見積システムの機能検証に用いたテスト環境、テストケースの定義、およびテストの実行結果（エビデンス）についてまとめた交付文書（提出物）です。

---

## 1. テスト実施概要

本システムでは、バックエンドのビジネスロジックおよびAPIについて自動化テスト（JUnit 5）を構築・実行し、フロントエンドおよびコンテナ連携について手動による受入シナリオテストを実行しました。

### テスト実行環境
* **バックエンドテスト**: JUnit 5, Spring Boot Starter Test, MockMvc (APIコントローラ検証用)
* **テスト用データベース**: H2 Database (PostgreSQL互換モードによるインメモリ動作。外部接続不要で100%ポータブルに再現可能)
* **フロントエンド・受入テスト**: Google Chrome 最新版を用いた実画面手動動作検証

## 3. 結合テスト仕様および実行結果 (IT)
